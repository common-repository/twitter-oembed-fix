This simple plugin fixes oembed for twitter. 

== Description ==

Twitter buttons are displayed in a language that is defined by server location.
Twitter OEmbed fix uses WPLANG variable or reverts to English by default
Activate it and forget about it

== Installation ==

1. Upload `twitter-oembed-fix` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Grab a beer and relax
